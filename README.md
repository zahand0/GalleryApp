# Gallery App

## Features:
### - asynchronous paginated image loading
### - camera preview
### - image capturing
### - runtime permissions
##
### Used libraries: Paging3, CameraX, Compose, Coil, Coroutines, Dagger-Hilt, Compose Navigation
